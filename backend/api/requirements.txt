# Minimal dependency set for the Vercel serverless deployment.
#
# Vercel's Python Serverless Functions have a hard deployment-size limit
# (commonly ~250MB unzipped). The full requirements.txt easily blows past
# that with shap's numba/llvmlite chain (~155MB) and scikit-learn+scipy
# (~145MB, only needed to unpickle/serve sklearn-wrapper models). This file
# drops both:
#   - shap              -> explainability.py already has a deterministic
#                          fallback when SHAP/the model artifact is absent
#                          (see explain_prediction's except branch)
#   - scikit-learn/scipy -> ml/train.py now saves native LightGBM/XGBoost
#                          Boosters (see income_engine.py / risk_engine.py /
#                          intent_engine.py), which predict without sklearn
#   - langgraph/langchain-core -> only used by the standalone
#                          agents/loan_advisor_graph.py demo script, not by
#                          the deployed FastAPI app
#   - google-genai       -> optional (offline advisor fallback covers chat);
#                          add back locally if you want live Gemini answers
#
# Result: ~190MB installed, vs. ~700MB+ with the full requirements.txt.
